# Optimization: Reduce Token Consumption

To minimize token usage and prevent "9600 token" prompt overhead:
1. **Title Generation:** Disabled. Do not attempt to automatically generate or update chat/issue titles unless explicitly asked.
2. **Follow-up Generation:** Disabled. Do not suggest follow-up questions or next steps unless the task requires it.
3. **Tags & Metadata:** Minimal. Only add tags if they are critical for organization.
4. **Autocomplete:** Do not provide unsolicited autocomplete suggestions.

Stick to the core task. Efficiency is priority.
